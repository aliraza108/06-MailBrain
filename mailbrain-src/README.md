# KAIRO Frontend

AI-powered autonomous email operations manager.

## Stack
- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- React Router
- Recharts + Lucide

## Development
```sh
npm install
npm run dev
```

## Environment
The frontend uses the live API at `https://06-mailbrain-api.vercel.app` via `src/lib/api.ts`.
